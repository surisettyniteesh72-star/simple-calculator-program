# Simple Calculator (Python)

A basic Python calculator that performs simple arithmetic operations based on user input.

## Features

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Invalid operator handling

## How to Run

```bash
python simple_calculator.py
```

Enter two integers and an operator (`+`, `-`, `*`, `/`) when prompted.

## Technologies

* Python 3
* Conditional Statements (`if-elif-else`)
* User Input (`input()`)
